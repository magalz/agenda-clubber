# Story 3.2: Filtros de Inteligência (Gênero/Região)

> **Status:** Done
> **Versão:** 1.0
> **Responsável:** River (SM)

## Story
**As a** Collective,
**I want** to filter the calendar view by Musical Genre and Region,
**so that** I can specifically analyze potential competition or collaboration opportunities within my niche and location.

## Acceptance Criteria
1. [x] A filtering bar in the Calendar Dashboard.
2. [x] Multi-select filter for `Genre` (using the shared `MUSICAL_GENRES` list). [Source: Story 2.4]
3. [x] Filter for `Region` (Fortaleza regions/neighborhoods).
4. [x] Toggle to show/hide "Conflicting Events" only (filtering for events that triggered warnings/conflicts).
5. [x] Filters must update the URL (search params) to allow sharing specific views.
6. [x] Filtering must be performed server-side or efficiently on the client to handle large datasets.
7. [x] UI remains clean and consistent with the industrial theme.
