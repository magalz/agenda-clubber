# Story 3.3: Detalhes e Contato para Arbitragem

> **Status:** Draft
> **Versão:** 1.0
> **Responsável:** River (SM)

## Story
**As a** Collective,
**I want** to see the contact details of another collective when there is a conflict,
**so that** I can reach out to them and negotiate an "arbitration" (date adjustment or lineup coordination) before anyone goes public.

## Acceptance Criteria
1. [ ] Detailed view of an event (modal or page) showing: Full Title, Description, Artists involved, and the owning Collective's name.
2. [ ] If the event is 'Identified' or 'Public', display a "Contact for Arbitration" button. [Source: prd.md#FR3]
3. [ ] Contact information must include: Instagram Handle and/or Website URL of the collective. [Source: Story 1.2]
4. [ ] If the event is 'Anonymous', the contact button must be HIDDEN or replaced by a "Request Identity" (optional future scope, for now, just hide).
5. [ ] Only authenticated collectives can see the contact info of other collectives.
6. [ ] UI reflects the industrial dark theme.
