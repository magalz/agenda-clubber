# Story 3.1: Grid View do Calendário (Mês/Semana)

> **Status:** Ready for Review
> **Versão:** 1.0
> **Responsável:** River (SM)

## Story
**As a** Collective,
**I want** to see all events (mine and competitors') in a calendar grid view (monthly and weekly),
**so that** I can have a clear visual understanding of the scene's schedule and plan my own events better.

## Acceptance Criteria
1. [x] A calendar dashboard at `/dashboard/calendar` showing events in a monthly grid. [Source: prd.md#FR2]
2. [x] Toggle between 'Month' and 'Week' views.
3. [x] Events must be color-coded by status (e.g., Confirmed = Green, Planning = Yellow, Idea = Zinc).
4. [x] Anonymous events must be displayed as "Reserved Slot" without leaking owner details. [Source: Story 2.2]
5. [x] Clicking an event in the grid should open a summary sidebar or tooltip with basic info (Title, Genre, Region, Start/End times).
6. [x] Navigation to previous/next month or week.
7. [x] Responsive design: mobile-friendly list view or simplified grid for smaller screens.
